abstract:
  This project is a simple weather inspecter, where you can click on the map and see what
  the current forecast is at that location. It was developed as an assignment for my
  study at Galvanize Inc. It features some 3d effects and CSS transitions.
---

## Tools Used

- [jQuery](https://jquery.com)
- [Three.js](https://threejs.org)
- [Leaflet.js](https://leafletjs.com)
- [Open Street Maps API](https://wiki.openstreetmap.org/wiki/API)
- [7Timer API](https://www.7timer.info/)
- [Sunrise/Sunset API](https://sunrise-sunset.org/)

## Techniques Employed

- [Linear Interpolation](https://en.wikipedia.org/wiki/Linear_interpolation)
- [Spherical Interpolation](https://en.wikipedia.org/wiki/Slerp)
- Asynchronous State Management
- ["Intended State"-Based Animation](https://github.com/athanclark/mcspb-frontend-project/blob/master/graphics.js#L262)
