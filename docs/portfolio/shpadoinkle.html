Shpadoinkle

Keywords:
Haskell, VirtualDOM, GHCJS
Development Date:
2023-11-14
Website:
https://shpadoinkle.org

Abstract

Shpadoinkle is a web application framework for developing both front-ends and back-ends in Haskell using GHCJS. It also has an agnostic virtual-dom for the front-end, letting users decide which engine works best for them.
