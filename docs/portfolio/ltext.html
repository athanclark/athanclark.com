位text

Keywords:
CLI, Templating, Text-Files, Functions, Lambda-Calculus
Development Date:
2023-11-13
Website:
https://ltext.github.io

Abstract

位text is a templating engine for text files, which lets you treat text files as functions - you can directly apply text files as the content of another. There is no preconfigured escape syntax; each file can define its own.
